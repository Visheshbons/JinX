# Welcome to JinX!

JinX will be a binary experimantal OS built by some high school students.
It will be built entirely by hand, with a custom kernel and custom programming languages.

## The Idea
The original idea was to build a ternary OS, with custom ternary software, but due to hardware limitations, we have downgraded to binary.
However, everything in the OS will be made from scratch, with a completely custom user interface and kernel.
This is **not** a Build Your Own Linux project, as we will not neccecarily be using Linux systems.
